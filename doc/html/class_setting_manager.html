§ readSettings()

void SettingManager::readSettings ( )

读取配置文件

实现时使用QSettings来读写配置，QSettings根据程序的的organizationName和 applicationName来获得文件位置。 所以主程序需要做如下设置： QCoreApplication::setOrganizationName("orgName"); QCoreApplication::setApplicationName("appName");

如果主程序中不做设置，那么tingUI提供默认值： organizationName = tinyUI-Dev applicationName = tinyUI
